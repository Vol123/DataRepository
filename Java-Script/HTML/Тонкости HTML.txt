SPAN
1. Строчные элементы часто используют для стилизации. 
Например, тег <span> используют для оборачивания участка текста. 
Потом этот участок стилизуют с помощью CSS

2. В большинстве случаев строчные элементы можно помещать внутрь блочных, 
а блочные нельзя помещать внутрь строчных. Исключение — ссылки. 
Внутрь ссылки можно помещать блочные элементы. 
Такое использование разрешили в стандарте HTML5
